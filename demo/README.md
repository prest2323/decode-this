# 📦 Demo Assets — Decode This v2

This folder contains the scripting, checklist, and asset documentation for the Decode This v2 live demo.

## 🌟 Hero Document
The primary hero document for the live walkthrough is the **SBA 7(a) Small Business Loan Application (Form 1919)**. 

### **Mock Flow Highlights:**
1. **🛡️ Protect view:** Instantly translates the 30-page document into a 3-sentence summary and highlights key liabilities: non-refundable fees, personal asset guarantees, background checks, and strict deadlines.
2. **📋 Checklist view:** Distinguishes on-page interactive inputs (address, legal name) from off-page prerequisites (tax returns, EIN filings).
3. **🎯 Guided canvas spotlights:** Groups related inputs (e.g. Street + City + State + Zip) into a single spotlight and positions instructions dynamically using collision-free coordinate math.
4. **💬 Contextual AI chat:** Directly answers step-specific questions about loan terminology.
5. **📥 High-fidelity Export:** Generates completed form packages in one click.
